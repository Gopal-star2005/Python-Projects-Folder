# 📘 Overview

The **Word Counter** is a Python-based command-line application that reads a text file and computes various statistics including:

- Word count
- Character count
- Character count without spaces
- Line count

It also provides a **preview of the file content**, making it a simple **text analysis utility**.

---

# 👤 About the Project

This project demonstrates:

- File handling in Python
- Text processing techniques
- Exception handling
- Basic file statistics computation
- CLI-based user interaction

The application can analyze existing files or generate a **sample text file for testing**.

---

# ✨ Key Features

- 📄 Count total words in a text file
- 🔤 Calculate total characters
- ✂️ Characters excluding spaces
- 📏 Count number of lines
- 👀 Preview first 20 words of the file
- 📁 Create sample text files for testing
- ⚠️ Robust file error handling

# 🚀 Workflow

1. User selects an option
2. Program either:
   - Reads an existing file
   - Creates a sample text file
3. File content is processed
4. Statistics are calculated
5. Results are displayed in a formatted output
